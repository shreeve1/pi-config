# Full Team

The full development team for complete software engineering workflows.
Includes the entire pipeline from exploration through implementation,
testing, security review, and documentation.

## Capabilities
- Codebase exploration and web research (scout, web-searcher)
- Planning and implementation (planner, builder)
- Code review and quality assurance (reviewer, tester)
- Security auditing (red-team)
- Debugging and root cause analysis (investigator)
- Documentation generation (documenter)

## When to Use
Use this team for general-purpose development work where the full
pipeline is needed. For specialized work, consider switching to a
focused team (info, frontend, qa, pi-pi).
